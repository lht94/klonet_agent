# Klonet 知识集合总览

这个目录保存 Mentor Agent 面向 Klonet 问答的 curated 知识文档。

## 路由原则

前置意图解析先判断用户问题的任务类型、操作阶段、目标组件和排除方向，再通过各目录下的 `manifest.json` 选择候选文档集合，最后在集合内做具体 chunk 检索。

## 当前集合

- `klonet_foundation`：项目概览、架构和术语。
- `klonet_environment_setup`：基础环境和依赖安装。
- `klonet_runtime_startup`：平台启动、停止和重启。
- `klonet_platform_usage`：普通用户浏览器使用平台。
- `klonet_topology_deploy`：拓扑部署流程。
- `klonet_vm_networking`：KVM、虚机、终端和组网。
- `klonet_satellite_platform`：卫星平台和天地一体化仿真。
- `klonet_common_troubleshooting`：通用故障排查。
- `klonet_development`：后端 API 开发和 Git 协作。
